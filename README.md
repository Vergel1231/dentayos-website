# DentAyos — Clinic Portal

A Filipino-built clinic management portal designed for dental and aesthetic clinics in the Philippines.

🌐 **Live site:** https://vergel1231.github.io/dentayos-website/

---

## Overview

DentAyos provides a complete four-portal solution connecting the clinic owner, clinic staff, and patients in one system.

| Portal | File | Access |
|---|---|---|
| Main Website | `index.html` | Public |
| Clinic Portal | `clinic.html` | Clinic staff only |
| Patient Portal | `patient.html` | Patients only |
| Owner Portal | `owner.html` | DentAyos owner only |

---

## Features

### Clinic Portal (`clinic.html`)
- Secure staff login with role-based accounts
- Dashboard with stats — total patients, today's appointments, reminders sent, no-shows
- Patient records management — add, view, and manage patients
- Appointments section — today's schedule and upcoming this week
- Reminders section — full log of sent reminders with send reminder modal
- View patient record modal with Send Reminder shortcut
- Collapsible sidebar for full-screen view

### Patient Portal (`patient.html`)
- Secure patient login — each patient has their own account
- Personalized welcome with next appointment highlighted
- View personal dental records
- Treatment history with procedure, dentist, and notes
- Reminders and notifications feed
- Interactive appointment booking with date and time picker
- Log out returns to login screen

### Owner Portal (`owner.html`)
- Restricted access — DentAyos owner only
- Dashboard with platform-wide stats — total clinics, monthly revenue, total patients, overdue payments
- Clinic accounts management — add, view, and monitor all registered clinics
- Revenue breakdown — per-clinic monthly income, paid vs overdue
- Activity log — all platform events (new signups, payments, alerts)

## Technology

Built with pure HTML5, CSS3, and JavaScript — no frameworks, no dependencies.

- Custom CSS with DM Sans and DM Serif Display fonts
- JavaScript for section switching, login logic, and dynamic rendering
- Intersection Observer for scroll animations (main website)
- Fully responsive for mobile and desktop

---

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser to view the main website
3. Navigate to `clinic.html` or `patient.html` to test the portals
4. The site is fully static — no server or build step required
