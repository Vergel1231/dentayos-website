# DentAyos - Clinic Portal

A Filipino-built clinic management portal designed for dental and aesthetic clinics in the Philippines.

## Overview

DentAyos provides a comprehensive solution for clinic management with two powerful sides:
- **Clinic Portal**: Everything staff needs to run the clinic — patient records, appointments, reminders
- **Patient Portal**: Professional self-service for patients to view records, book appointments, and receive reminders

## Features

### For Clinics
- Secure staff login and dashboard
- Patient record management
- Appointment scheduling and management
- Automated patient reminders
- Treatment history tracking

### For Patients
- Personal login portal
- View own medical records
- Online appointment booking
- Automated visit reminders
- Secure access anytime, anywhere

## Pricing

- **Pilot Plan**: ₱1,500/month (first 3 months), then ₱3,000/month for Metro Manila clinics
- **Standard Plan**: ₱3,000/month + ₱6,500 setup fee

## Technology

Built with:
- HTML5
- CSS3 (Custom styling with DM Sans and DM Serif Display fonts)
- JavaScript (Intersection Observer for animations)
- Responsive design for mobile and desktop

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. The website is static and ready to use
